LatLongConverter
================

LatLongConverter converts positions from geodetic system to Gauss-Krueger or UTM coordinates. This program is distributed as freeware for the operating systems Microsoft Windows, Apple OS X and Linux.

Developer and contact Rainer Sieger. The software is provided as freeware without warranty by the Alfred Wegener Institute for Polar and Marine Research (AWI).
